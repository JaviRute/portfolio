Spanish Vocab Trainer

This Python app contains all vocab for GCSE Spanish Higher, and was a great tool for my Y11 students when preparing for their tests.

The user can choose the topic by clicking in the 'Choose Module' tab. A random Spanish expression will appear, and he can opt for receiving up to two hints per expression.

Spanish Verbs Practice

Another app for my students, this one concentrates on Spanish regular verbs in 8 different tenses. It was extremely useful in my lessons.

The user can select the tense and the person (I, you, etc) he wants to practice. If the user provides 3 wrong answers, the program will eventually show the right answer.

The Chuck Norris Joke Translator

Created with Python (Tkinter) and using the Chuck Norris API and the Google Translate API.

It tells you a Chuck Norris fact and translates it to 12 languages, including Welsh.

Chuck Norris does not use a debugger. He stares down until the code confesses.

The Morse Code Translator: Street Fighter Edition

This extremely useful and potentially life-saving app takes the user input and converts it into morse code, using Street Fighter punch sounds.

Top 1000 IMDB Movies

In this data science project I used a json file with information about IMDB and Metacritic's best rated movies.

I created graphs about some of the movies to watch, actors which appear frequently, best directors, etc.

Between other charts, you will find a very handy movie picker (see image). The user simply needs to choose a year (and a genre if he wishes) and he will see a list of recommendations.

World Homicide

Another data science project where I studied homicide across countries and made charts which represent both total numbers and also average per inhabitant.

You can check the number of murders filtering by region, continent, country, year, etc. Very handy when planning for your next holidays!
